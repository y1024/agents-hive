## 1. 施工原则

- 不新建平行 `internal/run` 或 `internal/capability`。
- 继续复用 `internal/router`、`agentquality`、`observability`、`journal`、`trajectory`、`qualityworkbench`、`memory` 和 Admin API。
- `tool_search` 只返回发现和审计元数据，不改变 `RouteDecision.AllowedTools`。
- Admin 聚合可跨 owner，但响应必须走结构化脱敏，不返回 raw credentials。
- Memory explain 必须展示 owner/domain/source，但按当前 request runtime context 隔离跨 owner/domain 内容。
